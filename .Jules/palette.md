## 2024-05-06 - Implement Form Validation with Required Fields
**Learning:** Native form validation via the `required` and `aria-required="true"` attributes improves accessibility and provides immediate browser feedback to the user before custom JS validation triggers. This prevents screen readers from announcing visually required fields as optional.
**Action:** When creating forms, ensure visual indicators of requirement (like asterisks) are accompanied by semantic HTML `required` attributes and `aria-required` tags.
